# HealthDoc 用户功能说明

HealthDoc 是体检评价与健康档案系统。访客可以浏览公开机构与套餐，并通过注册页创建普通用户账号。登录和注册均使用一次性图片验证码。

普通用户登录后可以手工创建体检档案、录入体检指标，也可以上传 PDF、PNG、JPG、JPEG 或 WebP 报告进行 OCR。OCR 结果必须由用户核对并确认后，才会成为已确认档案。

用户可以添加亲友。只有亲友本人明确授权后，用户才能代为管理其档案、查看趋势或在单次同意后将相关档案用于健康 AI。授权撤销后，后续请求会立即失去访问权。

指标趋势页可以按照档案归属人和指标查看历史折线、最新值、最小值和最大值。健康 AI 只能提供指标科普和一般生活建议，不能诊断疾病、推荐处方药或替代急救和医生。

普通用户在健康 AI 中可以“精确选择档案”，也可以选择“检索该归属人的全部已确认档案”。两种方式都只对下一次请求生效，必须逐次确认同意；系统只把回答所需的最小必要指标事实发送给 DeepSeek，不会把私人档案写入公共知识索引。

评论体检机构前，用户必须在系统中拥有该机构的体检档案。评论提交后需要管理员审核。当前系统不提供自助找回密码，忘记密码需要联系人工客服。
